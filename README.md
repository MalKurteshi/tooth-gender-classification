# Tooth Gender Classification

This repository contains Python code to build a classification model that predicts gender based on tooth specifications.

## Project Overview

- **Objective:** Predict gender using features extracted from a tooth specifications dataset.
- **Language:** Python
- **Contents:** Data analysis, preprocessing, model training, and evaluation.

## Structure

```
/
├── data/                # Tooth specifications dataset
├── notebooks/           # Jupyter notebooks for analysis and modeling
├── src/                 # Python scripts (preprocessing, training, etc.)
├── requirements.txt     # Python dependencies
├── README.md            # Project documentation
└── .gitignore           # Ignore unnecessary files
```

## Getting Started

1. Clone the repo:
   ```bash
   git clone https://github.com/MalKurteshi/tooth-gender-classification.git
   cd tooth-gender-classification
   ```
2. Install dependencies:
   ```bash
   pip install -r requirements.txt
   ```
3. Place your dataset in the `data/` folder.
